Execution Instructions:


[CLI] Crawler

command: python3 scraper.py

This will show the progress of the crawler on the command prompt. Will
save the web scraper

And notify once the scraper is done with its job. The software is
packed with a saved dump of the web pages. you can directly run the
app.


[GUI] Search App:

Command: python3 main.py

This will run the search engine. User can type a query and get the
results. It will perform tf-idf and page rank computations.


Libraries used:

Tkinter

graph

json

regex

nltk

beautifulsoup
